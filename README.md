## Installation

Suspecting you're using Linux:

```sh
git clone https://github.com/makixx/nagakunai
cd nagakunai
npm install
cp settings.example.js settings.js
```
Edit `settings.js` and set the port.

Finally, run it!

```sh
node app.js
```
